StickyJS documentation



StickyJS is a small and powerfull javascript plugin for easy create sticky elements. Ajustable relative elements

How to use

To use StickyJs plugin use following code:

stickyJs('.sticky-widget').setOptions({relativeElement: '.content', topMargin: 46}).start();

StickyJs Options

{
	className: 'stickyJsElement',    // sticky element className
	staticClassName: 'sticky-fix',   // static sticky element className
	fixedClassName: 'sticky-fix',    // fixed sticky element className
	absoluteClassName: 'sticky-abs', // absolute sticky element className
	topMargin: 0,                    // margin top  if you have fixed top bar
	bottomMargin: 0,                 // margin bottom if you want 
	relativeElement: 'body'          // sticky relative element className or element
}
